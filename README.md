## ⌨️ 本地开发

```bash
$ git clone git@github.com:single-spa-react/single-spa-react-redux-toolkit.git
$ cd single-spa-react-redux-toolkit
$ npm install
$ npm start
```

打开浏览器访问 http://127.0.0.1:8080/project1/。
